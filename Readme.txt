ftprobot.py - исполняемый файл.
jillmodule.py - подгружаемые модули (должен находиться в одном каталоге с ftprobot.py)
config.cfg - конфигурационный файл списка заданий (должен находиться в одном каталоге с ftprobot.py)
Структура config.cfg:
par1.1;par1.2;...;par1.9;
par2.1;par2.2;...;par2.9;

par*.1 - полный путь к локальной директории
par*.2 - ftp хост
par*.3 - ftp порт
par*.4 - путь к директории на ftp сервере
par*.5 - ftp логин (в данной версии в открытом виде)
par*.6 - ftp пароль (в данной версии в открытом виде)
par*.7 - тип задания (0 - отправка; 1 - получение)
par*.8 - полный путь к сигнальному файлу
par*.9 - строка сигнального файла

proxy.cfg - конфигурационный файл прокси сервера (может находиться в одном каталоге с ftprobot.py, но не обязан... :))
Структура proxy.cfg:
par1;par2;par3;par4;par5;

par1 - тип прокси (в данной версии "http" или "sock5")
par2 - proxy хост
par3 - proxy порт
par4 - proxy логин (в данной версии в открытом виде)
par5 - proxy пароль (в данной версии в открытом виде)
